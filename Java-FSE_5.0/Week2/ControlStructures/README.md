# Control Structures (PL/SQL)

This folder contains self-contained PL/SQL solutions for:

1. Apply 1% discount to loan interest rates for customers above 60 years old.
2. Set `IsVIP` flag to TRUE for customers with balance over $10,000.
3. Print reminder messages for loans due within the next 30 days.

> Note: You may need to adjust table/column names to match your database schema.

## Files
- `scenario1_discount.sql`
- `scenario2_vip.sql`
- `scenario3_reminders.sql`
