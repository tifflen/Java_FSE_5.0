# Ecommerce Platform Search Function (Exercise 2)

## What this demonstrates
- `Product` model (productId, productName, category)
- Linear search (works on unsorted arrays)
- Binary search (requires sorted array by productId)
- Basic time-complexity expectations in comments/README

## Compile & Run (no build tool required)
From this folder:

```bat
javac *.java
java SearchTest
```

## Notes
- Binary search assumes `Product[]` is sorted in **ascending** order by `productId`.
